Q6 — Tiny calculator function

Write mini_calc(a, b) that returns a named numeric vector with: sum = a+b, diff = a-b, prod = a*b, quot = a/b (NA if b==0).

Return like c(sum=…, diff=…, prod=…, quot=…). Handle b==0 for quot.

mini_calc <- function(a, b) { c( sum = a + b, diff = a - b, prod = a * b, quot = if (b == 0) NA_real_ else a / b ) } 
mini_calc <- function(a, b) {
  c(
    sum  = a + b,
    diff = a - b,
    prod = a * b,
    quot = if (b == 0) NA_real_ else a / b
  )
}
