Q11 — One-sample hypothesis test on the variance (σ² known under H₀)

We want to test whether the variance of Track.Score is equal to a target value. Suppose a platform claims that Track.Score has variance \(\sigma_0^2 = 100\) (i.e. SD=10) We will test:

\((H_0: \sigma^2 = 100)\) vs \((H_1: \sigma^2 \neq 100)\)

This is a chi-square based variance test.

NoteInfo

For a one sample variance test \[\chi^2 = \frac{(n - 1)s^2}{\sigma_0^2} \sim \chi^2_{\,n - 1} \quad \text{under } H_0.\] EnvStats::varTest() does this for you and gives the p-value.
